# Exportar backup de tus bases de datos

> En la barra de menu, ir a "SERVER"
> Una vez allí seleccionar "DATA EXPORT"

> A continuación nos aparece una ventana con dos mitades para elegir las bases de datos a exportar (de lado izquierdo).
> Una vez seleccionadas las bases de datos, podemos seleccionar las tablas (de lado derecho).

> En la parte inferior, seleccionar el desplegable en la opción 
> "Dump Structure and data"

> En las opciones de exportacion (Exportr Options) debemos marcar el tilde que dice: Export to Self-contained file. Ahora podemos elegir el direcotrio de destino y adem´´as el nombre del archivo de destino. 

> Para finalizar, pulsamos el botón "Start Export"
